# Mbira Theory Map — Engine Reference

## The 7-Note Scale

Mbira uses a heptatonic (7-note) scale. Roots and fifths preferred.

## The 7 Chords

| Chord | Notes |
|-------|-------|
| Chord 1 | 1, 3, 5 |
| Chord 2 | 2, 4, 6 |
| Chord 3 | 3, 5, 7 |
| Chord 4 | 4, 6, 1 |
| Chord 5 | 5, 7, 2 |
| Chord 6 | 6, 1, 3 |
| Chord 7 | 7, 2, 4 |

Each chord lasts 4 beats (6 in Tadzungaira). 12 chords × 4 beats = 48-beat cycle.

## The 7 Progressions

- **Progression 1:** 1, 3, 5, 1, 3, 6, 1, 4, 6, 2, 4, 6
- **Progression 4 (most common):** 4, 6, 1, 4, 6, 2, 4, 7, 2, 5, 7, 2
- Progressions 2,3,5,6,7 = transposed versions of Progression 1

## 5-Factor Song Identity Model

1. Same notes on different tuning
2. Transposed notes for similar sound
3. Words sung
4. Specific melody line
5. Position of the beat
